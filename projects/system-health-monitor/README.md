# System Health Monitor

## Description
Automated system health monitoring script built with Bash.
Runs every hour via cron job and generates detailed reports.

## Features
- CPU Usage Monitoring
- RAM Usage Tracking
- Disk Space Check
- Internet Connectivity Check
- System Uptime Monitoring
- Automated Alerts for high usage
- Daily report generation

## Technologies Used
- Bash Scripting
- Linux Commands
- Cron Jobs (Task Automation)
- Git/GitHub

## How to Run
chmod +x health-monitor.sh
./health-monitor.sh
