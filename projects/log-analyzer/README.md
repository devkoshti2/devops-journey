# 📝 Log Analyzer & Cleaner

## Description
Automated log analysis tool built with Bash scripting.
Generates logs, analyzes them for errors/warnings, and auto-cleans old files.

## Scripts
- generate-logs.sh  → Fake log entries banata hai
- log-analyzer.sh   → Errors/warnings count karta hai + report banata hai
- log-cleaner.sh    → 7 din purane logs delete karta hai

## Technologies
- Bash Scripting
- Linux Commands (grep, find, wc, awk)
- Cron Jobs
- Git/GitHub

## How to Run
chmod +x *.sh
./generate-logs.sh
./log-analyzer.sh
./log-cleaner.sh
